# AI-Driven Toolpath Optimization Engine for FDM 3D Printing

An intelligent FDM slicing application that converts STL models into optimized G-code using Genetic Algorithm and A* Path Planning.

## Features
- STL file upload
- Layer slicing
- Genetic Algorithm toolpath optimization
- A* path planning
- Optimized G-code generation
- G-code visualization
- Print statistics
- Streamlit-based user interface

## Tech Stack
- Python
- Streamlit
- NumPy
- Shapely
- Matplotlib
- Plotly
- SciPy

## Workflow
STL Upload
→ Layer Slicing
→ Genetic Algorithm Optimization
→ A* Path Planning
→ G-code Generation
→ Visualization
→ 3D Printing

## Future Improvements
- Multi-nozzle optimization
- Support generation
- Adaptive layer heights
- ML-based print parameter prediction
